Description

LDAP Caller ID is an application for the Android platform which helps you to identify incoming calls by searching the caller's number in the LDAP directory.

From the principle, this application only works if the Wi-Fi, 3G or 4G data connection is available during the incoming call. The search results can be inaccurate if there is more then one person with the same phone number in the LDAP directory.

This application is Open Source, so you can contribute to the code if you are missing some feature or you have found some bug.
